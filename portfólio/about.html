Sobre mim

Sou desenvolvedor Back-End com experiência em APIs REST escaláveis e seguras, utilizando Java (Spring), Node.js (Express) e Python (Flask), além de trabalhar com bancos de dados SQL (PostgreSQL, MySQL) e NoSQL (MongoDB). Tenho forte experiência em microsserviços, cloud computing e Docker, aplicando boas práticas de segurança, performance e arquitetura.

Principais competências:
Desenvolvimento de APIs REST com Java (Spring) e Node.js (Express).
Modelagem e otimização de bancos de dados SQL e NoSQL.
Arquitetura de microsserviços e integração entre serviços.
Experiência em AWS e Docker para escalabilidade e gestão de ambientes.
Testes automatizados (JUnit, Mockito, Postman) e CI/CD para qualidade contínua.
Controle de versões com Git e boas práticas de segurança (JWT, OAuth).
